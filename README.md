## Objective

The objective of this lab is to understand and practice version control using Git and GitHub. This includes repository creation, committing changes, branching, merging, pull requests, forking repositories, and issue tracking.

## Tasks Performed

### Task 1: Git Setup

Configured Git username and email using Git commands.

### Task 2: Repository Creation

Created a local Git repository, added a README file, and committed the changes.

### Task 3: Push to GitHub

Created a GitHub repository and pushed the local repository to GitHub.

### Task 4: Branching and Merging

Created a new branch, added a file, committed changes, merged the branch into the main branch, and pushed updates.

### Task 5: Pull Request

Created a branch, made modifications, pushed changes, and merged them using a GitHub Pull Request.

### Task 6: Fork and Issue

Forked a public repository and created an issue for practice.
